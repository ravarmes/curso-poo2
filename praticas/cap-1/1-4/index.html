Práticas 1.4 — Binding e Propriedades Observáveis

O que são Properties e Bindings?

No JavaFX, Properties são wrappers observáveis para valores. Quando o valor muda, qualquer componente vinculado (bound) é automaticamente atualizado.

Source Property

slider.valueProperty()

Target Property

label.textProperty()

Property Tipo do Valor Uso Comum
SimpleStringProperty String Textos, labels
SimpleIntegerProperty int Contadores, índices
SimpleDoubleProperty double Sliders, valores numéricos
SimpleBooleanProperty boolean Checkboxes, estados

Vantagens do Binding:

  • ✅ Sincronização automática (sem listeners manuais)
  • ✅ Código mais limpo e declarativo
  • ✅ Reatividade: UI reflete modelo automaticamente

Tipos de Binding

1. Binding Unidirecional (bind)

Target sempre segue Source. Target não pode ser alterado diretamente.

// Label sempre mostra o valor do slider
label.textProperty().bind(
    slider.valueProperty().asString("%.0f")
);

2. Binding Bidirecional (bindBidirectional)

Ambos sincronizam. Alterar um altera o outro.

// TextField e Property sincronizados
StringProperty nome = new SimpleStringProperty();
textField.textProperty().bindBidirectional(nome);

3. Binding com Expressões

// Largura do progresso = slider / 100 * largura total
progressBar.progressProperty().bind(
    slider.valueProperty().divide(100)
);

4. Listener Manual

slider.valueProperty().addListener((obs, old, novo) -> {
    System.out.println("Valor mudou: " + novo);
});
