Node — Classe base de todos os elementos visuais no JavaFX. Possui propriedades como:

  • id — identificador único (CSS)
  • style — estilos inline CSS
  • visible — controla visibilidade

Principais Componentes de Entrada

Componente Descrição Exemplo de Código
Label Exibe texto não editável new Label("Olá!")
Button Botão clicável new Button("Clique")
TextField Campo de entrada de texto new TextField()
CheckBox Caixa de seleção new CheckBox("Aceito")
ComboBox Lista suspensa new ComboBox<>()

Exemplo Completo:

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.*;
import javafx.scene.layout.VBox;
import javafx.stage.Stage;

public class ExemploComponentes extends Application {
    @Override
    public void start(Stage stage) {
        Label label = new Label("Digite seu nome:");
        TextField campo = new TextField();
        Button btn = new Button("Enviar");
        
        VBox root = new VBox(10, label, campo, btn);
        stage.setScene(new Scene(root, 300, 150));
        stage.show();
    }
}
