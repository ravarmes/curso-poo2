Servidor Echo Completo

import java.io.*;
import java.net.*;

public class ServidorEcho {
    public static void main(String[] args) {
        try (ServerSocket server = new ServerSocket(12345)) {
            System.out.println("Servidor iniciado na porta 12345");
            
            while (true) {
                // Espera conexão
                Socket cliente = server.accept();
                System.out.println("Cliente conectado: " + 
                    cliente.getInetAddress());
                
                // Streams para comunicação
                BufferedReader in = new BufferedReader(
                    new InputStreamReader(cliente.getInputStream()));
                PrintWriter out = new PrintWriter(
                    cliente.getOutputStream(), true);
                
                // Echo: reenvia o que recebeu
                String msg;
                while ((msg = in.readLine()) != null) {
                    System.out.println("Recebido: " + msg);
                    out.println("Echo: " + msg);
                }
                
                cliente.close();
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Classes Principais

Classe Função
ServerSocket Escuta em uma porta, aceita conexões
Socket Representa conexão com cliente
InputStream Lê bytes do cliente
OutputStream Envia bytes para cliente
BufferedReader Lê linhas de texto (wrapper)
PrintWriter Escreve linhas de texto (wrapper)
